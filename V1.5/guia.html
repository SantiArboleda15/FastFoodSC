✅ Paso 1: Instalar XAMPP o WAMP (si no lo tienes)

¿Qué es? Es un servidor local que corre PHP y MySQL en tu computadora.

Opción A: XAMPP (Recomendado)

  1. Descarga desde: https://www.apachefriends.org/
  2. Instala el programa
  3. Abre XAMPP Control Panel
  4. Inicia Apache y MySQL
  5. Verifica que estén en verde ✅

Opción B: WAMP

  1. Descarga desde: http://www.wampserver.com/
  2. Sigue el asistente de instalación
  3. Inicia los servicios

✅ Paso 2: Crear la Base de Datos

Opción A: Usando phpMyAdmin (Visual)

  1. Abre tu navegador y ve a http://localhost/phpmyadmin
  2. Haz clic en "Nueva" o "New"
  3. Nombre de BD: fastfood_db
  4. Presiona Crear
  5. Haz clic en la BD creada
  6. Ve a la pestaña "SQL"
  7. Copia y pega el contenido del archivo database.sql
  8. Presiona "Ejecutar" ✅

Opción B: Desde terminal (Avanzado)

mysql -u root -p < database.sql

✅ Paso 3: Configurar la Conexión

El archivo conexion.php contiene la configuración. Verifica que sea:

$host = "localhost"; $user = "root"; $pass = ""; // Sin contraseña por defecto en XAMPP $database = "fastfood_db";
⚠️ IMPORTANTE (Para producción):
  • Cambia la contraseña de MySQL
  • Usa credenciales seguras en producción
  • NO dejes datos sensibles en el código

✅ Paso 5: Probar el Sistema

5.1 - Estructura de carpetas

Coloca todos los archivos en la carpeta correcta:

Sistema Ruta
XAMPP C:\xampp\htdocs\PROBANDO-PROBANDO\
WAMP C:\wamp\www\PROBANDO-PROBANDO\
Linux /var/www/html/PROBANDO-PROBANDO/

5.2 - Acceder al sistema

  1. Abre tu navegador
  2. Ve a: http://localhost/PROBANDO-PROBANDO/index.html
  3. Haz clic en "Registrarse"
  4. Completa el formulario ✅

5.3 - Verificar en phpMyAdmin

  1. Abre http://localhost/phpmyadmin
  2. Ve a fastfood_db → empleados
  3. Verifica que tu usuario aparezca en la tabla ✅

❌ Problemas Comunes

❓ "Error de conexión a base de datos"

  • Verifica que MySQL esté corriendo (XAMPP/WAMP)
  • Confirma que la BD fastfood_db existe
  • Revisa las credenciales en conexion.php

❓ "Error 404 - Archivo no encontrado"

  • Verifica la ruta completa en el navegador
  • Asegúrate de colocar los archivos en htdocs (XAMPP)
  • Los nombres de archivos son sensibles a mayúsculas en Linux

❓ "El registro no funciona"

  • Abre la consola del navegador (F12)
  • Verifica qué error muestra
  • Confirma que registrar.php existe
